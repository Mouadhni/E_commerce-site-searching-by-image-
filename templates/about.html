About Our Project

It's an e-commerce web application that allows users to search for products by uploading an image. Here are the key features:

Primary Objective

The main goal is to create a dynamic web page that enables customers to search for products by presenting an image of the product.

Expected Features

  • Interface for Image Upload: Users can upload an image of the product.
  • Product Display: Initial display of a random set of images representing different products.
  • Product Search: When an image is selected, display a list of similar products corresponding to that image.

Required Technologies

  • HTML5, CSS (Bootstrap) for web page design and style.
  • Python for backend development.
  • Pre-trained Image Classification Model on ImageNet to find similar products based on user-submitted images.

Backend Development

Backend development involves using Python to handle the user-submitted image and provide a list of similar products based on a pre-trained image classification model.

Model Training

We'll utilize pre-trained image classification models, such as those available on Keras, trained on the ImageNet database, to classify images and find similar products.

Useful References

  • ImageNet Database for training image classification models.
  • Examples of image classification using models trained on ImageNet available at Keras Applications.

Our main objective is to create a user-friendly interface that allows customers to search for products by uploading an image, utilizing a pre-trained model to classify images and provide a list of similar products.
